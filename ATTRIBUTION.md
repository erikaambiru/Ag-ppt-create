# Attribution Guidelines / 帰属表示ガイドライン

## English

### When Using This Software

If you use, modify, or distribute this software, you **MUST** include the following attribution:

#### Minimum Required Attribution

```
Based on Ag-ppt-create by aktsmm
https://github.com/aktsmm/Ag-ppt-create
License: CC BY-NC-SA 4.0
```

#### Recommended Attribution Formats

**In README.md or Documentation:**

```markdown
## Acknowledgments

This project uses [Ag-ppt-create](https://github.com/aktsmm/Ag-ppt-create) by aktsmm, licensed under CC BY-NC-SA 4.0.
```

**In Source Code Comments:**

```python
# Based on Ag-ppt-create by aktsmm
# https://github.com/aktsmm/Ag-ppt-create
# License: CC BY-NC-SA 4.0
```

**In Generated PowerPoint Slides:**

Add a footer or final slide with:

```
Created using Ag-ppt-create
https://github.com/aktsmm/Ag-ppt-create
```

### When Making Modifications

If you modify this software, you must:

1. **State that changes were made**
2. **Indicate what was modified** (general description is sufficient)
3. **Keep the original attribution**
4. **Use the same license (CC BY-NC-SA 4.0)**

Example:

```
Based on Ag-ppt-create by aktsmm
https://github.com/aktsmm/Ag-ppt-create
Modified by [Your Name] - Added support for X feature
License: CC BY-NC-SA 4.0
```

## 日本語

### 本ソフトウェアを使用する場合

本ソフトウェアを使用、改変、配布する場合、以下の帰属表示を**必ず**含めてください：

#### 最小限必要な表示

```
Based on Ag-ppt-create by aktsmm
https://github.com/aktsmm/Ag-ppt-create
License: CC BY-NC-SA 4.0
```

#### 推奨される表示形式

**README.md またはドキュメント内：**

```markdown
## 謝辞

本プロジェクトは aktsmm による [Ag-ppt-create](https://github.com/aktsmm/Ag-ppt-create) を使用しています（CC BY-NC-SA 4.0 ライセンス）。
```

**ソースコードのコメント：**

```python
# Ag-ppt-create by aktsmm をベースにしています
# https://github.com/aktsmm/Ag-ppt-create
# License: CC BY-NC-SA 4.0
```

**生成された PowerPoint スライド内：**

フッターまたは最終スライドに以下を追加：

```
Ag-ppt-create を使用して作成
https://github.com/aktsmm/Ag-ppt-create
```

### 改変を加える場合

本ソフトウェアを改変する場合、以下が必要です：

1. **変更を加えた旨を明記する**
2. **何を変更したかを示す**（概要で十分です）
3. **元の帰属表示を保持する**
4. **同じライセンス（CC BY-NC-SA 4.0）を使用する**

例：

```
Based on Ag-ppt-create by aktsmm
https://github.com/aktsmm/Ag-ppt-create
Modified by [あなたの名前] - X機能のサポートを追加
License: CC BY-NC-SA 4.0
```
